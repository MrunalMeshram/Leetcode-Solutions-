Rotate An Array By K
Medium
Acceptance: 45%
Given an array of integers and an integer k, rotate the array to the right by k steps.

Examples:
Example 1:

Input: [1, 2, 3, 4, 5], 2
Output: [4, 5, 1, 2, 3]
Explanation: Rotate the array [1, 2, 3, 4, 5] to the right by 2 steps to get [4, 5, 1, 2, 3].
Example 2:

Input: [-1, -100, 3, 99], 3
Output: [-100, 3, 99, -1]
Explanation: Rotate the array [-1, -100, 3, 99] to the right by 3 steps to get [-100, 3, 99, -1].
